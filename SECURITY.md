# Security Policy

## Supported Versions
This project is under active development. Security fixes will be provided for the latest version on the main branch.

## Reporting a Vulnerability
If you discover a security vulnerability, please do NOT open a public GitHub issue.

Instead, email the maintainer with:
- A description of the issue
- Steps to reproduce
- Impact assessment (what data could be affected)

We will respond as soon as possible.

## Sensitive Data Warning
This system is intended to handle sensitive student data. Do not deploy it for real users until security hardening is complete and the project explicitly declares production readiness.
